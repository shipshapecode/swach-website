# Shortcuts

## macOS

* Open Picker `⌃` `⌥` `⌘` `P`
* Open Contrast Checker `⌃` `⌥` `⌘` `C`
* Undo Action `⌘` `Z`
* Redo Action `⇧` `⌘` `Z`

## Windows + Linux

* Open Picker `⌃` `⌥` `⊞ Win` `P`
* Open Contrast Checker `⌃` `⌥` `⊞ Win` `C`
* Undo Action `CTRL` `Z`
* Redo Action `⇧` `CTRL` `Z`
