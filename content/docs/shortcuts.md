# Shortcuts

## macOS

* Open Picker `⌃` `⌥` `⌘` `P`
* Open Contrast Checker `⌃` `⌥` `⌘` `C`
* Undo Action `⌘` `Z`
* Redo Action `⇧` `⌘` `Z`

## Windows + Linux

* Open Picker `Ctrl` `Alt` `⊞ Win` `P`
* Open Contrast Checker `Ctrl` `Alt` `⊞ Win` `C`
* Undo Action `Ctrl` `Z`
* Redo Action `⇧` `Ctrl` `Z`
